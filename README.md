# Dr.COM 校园网助手 (Dr.com-Windows-client)

> 💡 **声明**：本项目的核心代码由作者（飞行雪绒）独立构思与调试，由 AI 辅助生成。

一款专为 Windows 系统打造的 **Dr.COM 校园网极速登录客户端**。基于 Python + PyQt6 开发，告别每次开机都要打开浏览器手动输入账号密码的烦恼，将网络连接变成真正的“无感体验”。

-----

## ✨ 核心特性

  * **🚀 极速断线秒连**：后台静默守护，每 5 秒自动检测网络连通性。一旦发现掉线，毫秒级自动回拨，游戏/网课不掉线。
  * **📡 智能线路嗅探**：支持多运营商（移动 `@cmcc`、电信 `@telecom`、联通 `@unicom` 及校园内网），首次登录失败会自动遍历尝试所有后缀。
  * **🛡️ 单实例防多开**：底层调用 Windows `CreateMutexW` 互斥锁，确保后台只运行一个守护进程，绝不占用多余系统资源。
  * **🤖 贴心错误翻译**：内置“Dr.COM 黑话翻译机”，告别看不懂的 `userid error2`，精准提示“密码错误”、“账号欠费”或“异地登录”等异常情况并弹出警告拦截。
  * **🖥️ 无缝系统集成**：
      * 支持写入 Windows 注册表实现**开机静默启动**（静默挂载托盘）。
      * 最小化至系统托盘，不霸占任务栏空间。
      * 精美的 PyQt6 实时数据面板（在线时长、内网 IP、本机 MAC 实时显示）。
        

## 📥 下载与使用

### 对于普通用户（推荐）

1.  前往本仓库的 **[Releases](https://www.google.com/search?q=%23)** 页面。
2.  下载最新版本的 `DrCom校园网助手.exe`。
3.  双击运行，输入您的学号/账号及密码。
4.  勾选 **“断线秒连”** 和 **“开机启动”**，点击 **“开启极速连接”** 即可。
      * *关闭主窗口后，程序会自动隐藏在右下角系统托盘中为您持续护航。*



-----

## ⚙️ 配置文件说明

本程序会将您的登录配置信息（不含敏感网络劫持）保存在 Windows 的 `AppData` 目录下：

  * **路径**: `C:\Users\用户名\AppData\Roaming\DrComHelper\config.json`
  * **注意**: 密码以明文形式保存在本地，请勿将您的 `config.json` 文件发送给他人。
